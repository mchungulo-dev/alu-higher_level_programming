The ``MyList`` class
====================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Normal case:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [1, 2, 4]
    >>> print(my_list)
    [1, 4, 2]

Test present: check print_sorted() with no sorted append with negative number:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-4)
    >>> my_list.append(-2)
    >>> my_list.print_sorted()
    [-4, -2, -1]

Test present: check print_sorted() with empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
