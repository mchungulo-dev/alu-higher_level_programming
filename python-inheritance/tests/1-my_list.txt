The ``MyList`` class
====================

Import the class from the file:
    >>> MyList = __import__('1-my_list').MyList

Create an instance and add integers:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

Check the string representation of the list (unsorted):
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Check the sorted print:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Verify that the original list remains in the same order:
    >>> print(my_list)
    [1, 4, 2, 3, 5]
